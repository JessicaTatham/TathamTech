# LaRue PR — Design System

## Color Palette (6 colors only)

| Name | Hex | Usage |
|---|---|---|
| Soft Gray | #EEE5D3 | Page background, text on dark sections |
| Black | #151515 | Dark section backgrounds, primary text on light |
| Blue | #757CF3 | Hero background, brands section, stats section |
| Red Orange | #FF6C2D | Reserved (not on homepage currently) |
| Yellow | #FCB232 | Awards section background |
| Green | #2AA963 | Reserved (not on homepage currently) |

No other colors are used. Accent text on dark backgrounds uses Soft Gray italic — not a separate purple.

## Typography

**Two fonts:**
- **GT America Medium** — headings (if available; falls back to Garnett Medium)
- **Garnett Regular** — body copy

**Hierarchy (consistent across all sections):**

| Element | Font | Weight | Size | Style |
|---|---|---|---|---|
| Eyebrow labels | Garnett Medium | 500 | 0.75rem | Uppercase, tracked 0.25em |
| Section headline (H2) | Garnett Medium | 500 | clamp(1.75rem → 3rem) | Uppercase |
| Card/item title (H3) | Garnett Medium | 500 | clamp(1.05rem → 1.25rem) | Uppercase |
| Body copy | Garnett Regular | 400 | clamp(0.9rem → 1rem) | Normal |
| Featured quote | Garnett Regular | 400 | clamp(1.1rem → 1.25rem) | Normal, larger body |
| Button text | Garnett Medium | 500 | 0.95rem | Normal |
| Small/meta text | Garnett Regular | 400 | 0.75–0.85rem | Uppercase tracked |

**Rules:**
- Every H2 on the page is the same size (clamp 1.75rem → 3rem)
- Every H3 on the page is the same size (clamp 1.05rem → 1.25rem)
- Body copy is always the same size regardless of section
- Italic is used ONLY for accent words within a heading (same color as surrounding text, never a different color)
- No bold within body copy
- Eyebrows are always the same size and tracking
